글자 확대, 축소 버튼만들고 이벤트 핸들러 메서드를 이용해 기능을 구현

요구사항

  • 글자크기 : 8px(최소), 14px(기본), 20px(최대)
  • 확대 또는 축소 버튼클릭시 +,- 1px씩 확대/축소 기능
  • 100% 버튼 : 기본값 설정(14px)
  • 설정된 글자 크기에 맞춰 문장의 글자 크기를 변경
